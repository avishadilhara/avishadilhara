### Hi, I'm Avisha 👋

I'm a Computer Science graduate (First Class Honours, University of Westminster) working on **machine learning and NLP**, with hands-on experience fine-tuning Vision-Language Models for low-resource language problems.

- 🔬 My final year research fine-tuned **DeepSeek-OCR** and **LightOnOCR-2-1B** using LoRA/QLoRA to reach a **1.05% Character Error Rate** on printed Sinhala text — outperforming Google Document AI, Surya-OCR, and Tesseract v5.
- 📄 Paper accepted at **MERCon 2026** — [read it on arXiv](https://arxiv.org/abs/2606.29378)
- 🛠️ Also build full-stack apps and automation tooling (Next.js, Spring Boot, Selenium, AWS Lightsail)
- 🎤 Active Toastmasters member — Vice President Public Relations, Moratuwa Toastmasters Club
- 🎯 Currently looking for **ML/AI Engineering** roles in NLP, document understanding, or low-resource language tech
